In this document we'll go through examples of using AMF 5 from the [Examples Repository](https://github.com/aml-org/examples),
where you can clone, run, and test all AMF 5 examples in this documentation.

## Cover the basics
If you don't know how to use AMF yet, refer to the [Using AMF](../AMF/Using%20AMF/amf_setup) section to learn how to
[add AMF to your project](../AMF/Using%20AMF/amf_setup) and how to use AMFs [configuration and client](../AMF/Using%20AMF/amf_configuration) objects.
